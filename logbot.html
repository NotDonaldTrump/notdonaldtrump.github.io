LogBot is a Discord server bot which logs messages sent by members, including attachments.

LogBot is against Discord's TOC unless you have gained consent from every server member. Use at your own risk, I am NOT responsible for any damages caused.

Only members with admin rights will be able to access the log via the >log command which will be sent to them via DM.

V2.0.2

Installation:

  1. Download from here.
  2. Extract the zip contents (password can be aquired for free here)
  3. Read the README.txt file before starting!
  4. Run the setup script and wait for the window to close
  5. Open the config.js file and change the bot token and server name.
  6. Save the config.js file and close it.
  7. Run the start script!

Log files are found in /Logs/

Help/Questions/Requests can be directed here

Happy logging!
